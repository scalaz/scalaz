scalaz.iteratee

StepT

sealed trait StepT[E, F[_], A] extends AnyRef

The current state of an Iteratee, one of:

  • cont Waiting for more data
  • done Already calculated a result
E

The type of the input data (mnemonic: Element type)

F

The type constructor representing an effect. The type constructor scalaz.Id is used to model pure computations, and is fixed as such in the type alias scalaz.Step.

A

The type of the calculated result
